
Источник вдохновения: рыбы, которые умеют менять цвет в зависимости от фона.

Несколько вариантов:
1. 'main' - базовый вариант без мимикрии с drug-and-drop.
2. 'current' - рыба с мимикрией и с drug-and-drop. 
3. 'animation' - анимированная рыба со сменой цвета. 

Как пользоваться: double-click по html файлу в каждой из папок.
