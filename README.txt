Рыба, которая маскируется и принимает цвет в зависимости от фона.

Несколько вариантов:
1 'main' - базовый вариант без мимикрии с drug-and-drop
2 'current' - рыбка с мимикрией с drug-and-drop. Смена цвета неидеальная.
3 'js-анимация' - рыба ходит по маршруту.
